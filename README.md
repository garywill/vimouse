# ViScreen

Use keyboard to click anywhere on screen. OpenCV based. 

- Dependencies:
  - python3
  - opencv
  - tkinter
  - pillow (with pillow-tk)
  - numpy
  - pynput


**Usage**:

1. `ctrl + meta` to find and show keys (meta key = win key = super key)
2. Press some keys to move mouse to a position
3. `meta + ctrl` to click
